Understanding EJS Templates for Dynamic Web Development

EJS (Embedded JavaScript) is a templating engine that lets you generate HTML markup with plain JavaScript. This is particularly useful for dynamic web applications. In this guide, we’ll explore the basics of EJS and how to implement it effectively.

Setting Up EJS

const express = require('express');
const app = express();

app.set('view engine', 'ejs');

app.get('/', (req, res) => {
    res.render('index', { title: 'Home' });
});

app.listen(3000, () => {
    console.log('Server is running on http://localhost:3000');
});

First, ensure you have Express and EJS installed:

npm install express ejs

Basic EJS Syntax

EJS allows you to embed JavaScript logic directly in your HTML. Here’s a simple example:

<h1><%= title %></h1>

The <%= %> syntax is used for outputting values, while <% %> allows for control flow (e.g., loops and conditionals).

Using EJS with Data

To render a list of items dynamically, you can pass an array to your EJS template:

app.get('/items', (req, res) => {
    const items = ['Apple', 'Banana', 'Cherry'];
    res.render('items', { items });
});

In your items.ejs file, you could use:

<% items.forEach(item => { %>
    <p><%= item %></p>
<% }) %>

Conclusion

Using EJS templates makes it easy to create dynamic and interactive web applications. With EJS, you can leverage JavaScript to manage your HTML more effectively. For more details, refer to the EJS Documentation. Happy coding!
