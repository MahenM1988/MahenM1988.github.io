How to Set Up a Simple Node.js Server with Random Tokens

Step 1: Install Node.js

Make sure you have Node.js installed on your system. You can download it from the official Node.js website.

Step 2: Create a Project Directory

mkdir my-node-server
cd my-node-server

Step 3: Initialize a Node.js Project

Run the following command to create a package.json file:

npm init -y

Step 4: Install Dependencies

Install Express and dotenv:

npm install express dotenv

Step 5: Create a .env File

Create a file named .env in your project root:

touch .env

You can add any environment variables here, like the port number:

PORT=3000

Step 6: Create the Server File

Create a file named server.js:

touch server.js

Open server.js in your favorite text editor and add the following code:

require('dotenv').config();
const express = require('express');
const crypto = require('crypto');

const app = express();
const PORT = process.env.PORT || 3000;

// Middleware to generate a random session token
app.use((req, res, next) => {
    req.session = req.session || {};
    if (!req.session.token) {
        req.session.token = crypto.randomBytes(16).toString('hex'); // Generate a random token
    }
    next();
});

// Serve static files
app.use(express.static('public'));

// Route to get session token
app.get('/token', (req, res) => {
    res.json({ token: req.session.token });
});

app.listen(PORT, () => {
    console.log(`Server is running on http://localhost:${PORT}`);
});

Step 7: Create the HTML Page

Create a directory named public and an index.html file inside it:

mkdir public
touch public/index.html

Add some basic HTML content to index.html:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Node.js Server</title>
</head>
<body>
    <h1>Welcome to My Node.js Server!</h1>
    <p>Your session token is: <span id="token"></span></p>
    <script>
        fetch('/token')
            .then(response => response.json())
            .then(data => {
                document.getElementById('token').innerText = data.token;
            });
    </script>
</body>
</html>

Step 8: Start the Server

Run the server with the following command:

node server.js

Open your browser and navigate to http://localhost:3000 to see your page!
