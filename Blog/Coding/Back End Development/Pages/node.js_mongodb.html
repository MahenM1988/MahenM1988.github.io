How to Set Up a Simple Node.js Server with MongoDB

Step 1: Install Node.js

Make sure you have Node.js installed on your system. You can download it from the official Node.js website.

Step 2: Create a Project Directory

mkdir my-node-server
cd my-node-server

Step 3: Initialize a Node.js Project

Run the following command to create a package.json file:

npm init -y

Step 4: Install Express and MongoDB Driver

Install Express and the MongoDB driver:

npm install express mongodb

Step 5: Create the Server File

Create a file named server.js:

touch server.js

Open server.js in your favorite text editor and add the following code:

const express = require('express');
const { MongoClient } = require('mongodb');

const app = express();
const PORT = 3000;
const uri = 'YOUR_MONGODB_CONNECTION_STRING';

app.use(express.static('public'));

MongoClient.connect(uri, { useNewUrlParser: true, useUnifiedTopology: true })
    .then(client => {
        console.log('Connected to Database');
        const db = client.db('your_database_name');
        
        // Example route to fetch data from MongoDB
        app.get('/data', (req, res) => {
            db.collection('your_collection_name').find().toArray()
                .then(results => {
                    res.json(results);
                })
                .catch(error => console.error(error));
        });
    })
    .catch(error => console.error(error));

app.listen(PORT, () => {
    console.log(`Server is running on http://localhost:${PORT}`);
});

Step 6: Create the HTML Page

Create a directory named public and an index.html file inside it:

mkdir public
touch public/index.html

Add some basic HTML content to index.html:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Node.js Server</title>
</head>
<body>
    <h1>Welcome to My Node.js Server!</h1>
    <p>This is served from the server.</p>
    <div id="data"></div>
    <script>
        fetch('/data')
            .then(response => response.json())
            .then(data => {
                const dataDiv = document.getElementById('data');
                data.forEach(item => {
                    const p = document.createElement('p');
                    p.textContent = JSON.stringify(item);
                    dataDiv.appendChild(p);
                });
            });
    </script>
</body>
</html>

Step 7: Set Up MongoDB

If you don't have MongoDB installed, you can use MongoDB Community Server or MongoDB Atlas for a cloud database.

Make sure to replace YOUR_MONGODB_CONNECTION_STRING in server.js with your actual MongoDB connection string.

Step 8: Sample Database Structure

Here is a sample structure for your MongoDB database:

Database: your_database_name
Collection: your_collection_name
Documents:
{
    "_id": ObjectId("..."),
    "name": "Example Item",
    "description": "This is an example item.",
    "price": 29.99,
    "category": "example"
}

You can customize the fields in your documents as per your requirements. For instance, you can add fields like createdAt or updatedAt for timestamping.

Step 9: Customize for Your Use Case

To customize the database structure:

  1. Connect to your MongoDB instance using a MongoDB client (e.g., MongoDB Compass or the command line).
  2. Create a database with the name you specified in server.js.
  3. Create a collection with the name you specified in server.js.
  4. Add documents to your collection following the sample structure, adapting fields as necessary.

Step 10: Start the Server

Run the server with the following command:

node server.js

Open your browser and navigate to http://localhost:3000 to see your page!
