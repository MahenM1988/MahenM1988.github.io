Setting Up a Simple Flask Server

Step 1: Install Flask

First, ensure you have Python installed. Then, you can install Flask using pip. Open your terminal or command prompt and run:

pip install Flask python-dotenv

Step 2: Create the Project Directory

Create a directory for your Flask project:

mkdir my_flask_app
cd my_flask_app

Step 3: Create the Flask Application

Inside your project directory, create a new file called app.py and add the following code:

from flask import Flask, render_template, session
import os
from dotenv import load_dotenv
import secrets

# Load environment variables from .env file
load_dotenv()

app = Flask(__name__)
app.secret_key = os.getenv("SECRET_KEY", secrets.token_hex(16))  # Use secret key from .env or generate a random one

@app.route('/')
def home():
    # Generate a random session cookie for each user and tab
    session['user_session_id'] = secrets.token_hex(16)
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)

Step 4: Create the .env File

Create a file named .env in your project directory. This file can be used to store environment variables. Add your variables like this:

FLASK_ENV=development
SECRET_KEY=your_secret_key_here

Step 5: Create the HTML Template

Next, create a folder named templates in your project directory. Inside this folder, create a file named index.html and add some basic HTML:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Welcome to Flask!</title>
</head>
<body>
    <h1>Hello, Flask!</h1>
    <p>This is your index page.</p>
    <p>Your session ID is: <span><script>document.write(sessionStorage.getItem('user_session_id'));</script></span></p>
</body>
</html>

Step 6: Run the Flask Server

Now, you can run your Flask application. In your terminal, ensure you are in the project directory and run:

python app.py

Your Flask server should now be running on http://127.0.0.1:5000. Open this URL in your web browser to see your index page!

Conclusion

You have successfully set up a simple Flask server to serve an HTML page, used a .env file to manage environment variables, and implemented random session cookies for each user and tab. You can expand upon this basic setup to create more complex applications!
