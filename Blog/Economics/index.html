Using Diskpart to Format a USB Drive

Accessing Command Prompt Accessing Diskpart List Available Disks Identify Disk to Format Select Disk Clean Create Partition Primary Select Partition Primary Format Assign Drive Letter Exit Diskpart Exit Command Prompt
Accessing Command Prompt

  • Press Windows + X on your keyboard to open the Power User menu. Select "Command Prompt (Admin)" or "Windows PowerShell (Admin)".

    • Accessing Diskpart

  • In the Command Prompt or PowerShell window, type diskpart and press Enter.

    • List Available Disks

  • Type list disk and press Enter to display a list of all available disks on your computer.

    • Identify Disk to Format

  • Carefully identify your USB drive from the list. Note the size of the drive to avoid formatting the wrong disk.

    • Select Disk

  • Type select disk X (replace X with the number corresponding to your USB drive) and press Enter.

    • Clean

  • Type clean and press Enter. This command removes all partitions and volumes from the selected disk.

    • Create Partition Primary

  • Type create partition primary and press Enter. This command creates a new primary partition on the USB drive.

    • Select Partition Primary

  • Type select partition 1 and press Enter.

    • Format

  • Type format fs=ntfs and press Enter to format the partition using the NTFS file system. If you prefer FAT32, use format fs=fat32 instead.

    • Assign Drive Letter

  • Type assign letter=X (replace X with the desired drive letter) and press Enter. This assigns a drive letter to the USB drive.

    • Exit Diskpart

  • Type exit and press Enter to leave the DiskPart utility.

    • Exit Command Prompt

  • Type exit and press Enter to close the Command Prompt.
