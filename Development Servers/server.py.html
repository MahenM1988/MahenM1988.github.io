Setting Up a Simple Flask Server

Step 1: Install Flask

First, ensure you have Python installed. Then, you can install Flask using pip. Open your terminal or command prompt and run:

pip install Flask python-dotenv flask-cors flask-talisman flask-wtf

Step 2: Create the Project Directory

Create a directory for your Flask project:

mkdir my_flask_app
cd my_flask_app

Step 3: Create the Flask Application

Inside your project directory, create a new file called app.py and add the following code:

import os
import random
import string
import ssl
from flask import Flask, request, jsonify, session
from flask_cors import CORS
from flask_talisman import Talisman
from flask_wtf.csrf import CSRFProtect

app = Flask(__name__)

# Generate a random secret key
app.config['SECRET_KEY'] = ''.join(random.choices(string.ascii_letters + string.digits, k=32))

# Configure session cookie settings
app.config['SESSION_COOKIE_HTTPONLY'] = True
app.config['SESSION_COOKIE_SAMESITE'] = 'Lax'

# CORS configuration
CORS(app, origins=["https://your-frontend-domain.com"], supports_credentials=True)

# Security headers
Talisman(app)

# CSRF protection
csrf = CSRFProtect(app)

# Middleware to generate session tokens
@app.before_request
def generate_session_token():
    if 'token' not in session:
        session['token'] = generate_session_token_value()

def generate_session_token_value():
    return ''.join(random.choices(string.ascii_letters + string.digits, k=32))

# Route to get CSRF token
@app.route('/csrf-token', methods=['GET'])
def csrf_token():
    return jsonify({'csrfToken': csrf.generate_csrf()}), 200

# Example route to handle incoming data
@app.route('/api/data', methods=['POST'])
def handle_data():
    input_data = request.json.get('inputData')
    return jsonify({'message': 'Data received successfully'}), 200

# SSL configuration
ssl_context = ssl.SSLContext(ssl.PROTOCOL_TLS)
ssl_context.load_cert_chain('ssl/cert.pem', 'ssl/key.pem')

if __name__ == '__main__':
    port = int(os.getenv('PORT', 3000))
    app.run(ssl_context=ssl_context, host='0.0.0.0', port=port)

Step 4: Create the SSL Certificates

You need to have SSL certificates to run your server securely. Place your cert.pem and key.pem files in an ssl directory within your project folder.

Step 5: Run the Flask Server

Now, you can run your Flask application. In your terminal, ensure you are in the project directory and run:

python app.py

Your Flask server should now be running on https://127.0.0.1:3000. Open this URL in your web browser to access your API!

Conclusion

You have successfully set up a simple Flask server with CSRF protection, session management, and SSL. You can expand upon this basic setup to create more complex applications!
